# Soldat Website

Built with [GatsbyJS](https://www.gatsbyjs.org/)

## Installation & running locally

Clone the repository and run the following commands:

```sh
cd SoldatWebsite

npm install
npm start # http://localhost:8000/
```

## Build

```ssh
npm run build
```
